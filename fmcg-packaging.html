FMCG Packaging

I had to redesign the visual aspect of an existing fast moving consumer good package or fast moving good packaging range. I needed to redesign the product logo. I could base the package shape/structure on an existing package or template or I could completely design the shape/ structure of the package itself. I had to include a creative barcode designed by me, using an EAN13 barcode numbers that I had been allocated. The package was to be produced using 4 process colours (CMYK). Special or spot colours could be included and if used would need to be presented correctly in the digital file. I needed to represent the dieline correctly in my digital file. I needed to use imagery, typography, colour and supporting design elements that would help convey the message in the packaging and appeal to my chosen target audience.
